    Instalar NodeJs y NPM en MacOS

    Written by José on 20 Feb 2022

    Para instalar NodeJs y NPM en Mac necesitamos el gestor de paquetes Homebrew. Si no disponemos de él lo instalaremos escribiendo el siguiente código en la Terminal:

    /bin/bash -c “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)”

    Una vez dispongamos del Homebrew procedemos a instalar NodeJs y NPM escribiendo en la Terminal lo siguiente:

    brew install node

    Para comprobar que todo ha ido bien y la versión instalada utilizaremos:

    node -v ó npm -v

    Más comandos …

    Para actualizar NodeJs:

    brew update

    brew upgrade node

    Para desinstalar NodeJs

    brew uninstall node

    Y esto es todo lo que quería contar :)
